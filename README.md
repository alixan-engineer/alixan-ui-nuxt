# Alixan UI

Minimal Nuxt UI components with a small shadcn-like CLI.

## Usage

```bash
npx alixan-ui-nuxt add button
```

The CLI copies component source files into your project:

```txt
components/ui/Button.vue
utils/cn.ts
```

## Local Development

```bash
npm install
npm run dev
```

## Publish

```bash
npm version patch
npm publish --access public --otp=123456
```
